**Job Position API**

**Description**
Job Position API is a RESTful API designed to manage job position records, including salary, in a system. 
The API allows users to perform operations such as creating, retrieving, updating, and deleting job position data.
The project is documented using Swagger (OpenAPI), which provides an interactive interface for testing and exploring the API endpoints.

**Features**
* Create new job positions
* Retrieve all job positions
* View specific job position details
* Update job position information
* Delete job positions
* Interactive API documentation using Swagger

**Technologies Used**
* PHP
* MySQL
* Swagger / OpenAPI
* JSON
* Apache / XAMPP

**Installation / Setup**
1. Clone or download this repository.
2. Place the project folder inside the `htdocs` directory (if using XAMPP).
3. Import the database file into MySQL using phpMyAdmin.
4. Configure the database connection in the project files.
5. Start Apache and MySQL using XAMPP.
6. Open the Swagger documentation in your browser to explore the API endpoints.
